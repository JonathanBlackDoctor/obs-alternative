# SilentStream 사용자 안내서

OBS 없이 **PC를 켜면 자동으로 YouTube 라이브 송출 + 로컬 백업 녹화**가 시작되는
백그라운드 프로그램입니다. (Windows 10/11 x64 전용)

> ⚠️ **본인 소유 PC의 자기 기록·방송 목적 전용**입니다. 타인의 화면·음성을 동의
> 없이 캡처하는 용도로 사용할 수 없습니다 (EULA 참조).

## 1. 설치

1. `SilentStream-Setup-x.y.z.exe` 실행.
2. EULA(사용 동의) 확인 → 동의.
3. **자동 시작 방식** 선택:
   - **시작 프로그램(권장)** — 로그인 후 실행.
   - **작업 스케줄러** — 최고 권한, 로그인 직후 실행.
4. **녹화 폴더** 선택 (기본: `내 비디오\SilentStream`, 용량 큰 드라이브 권장).
5. 최초 실행 시 화면·오디오 캡처 동의 화면이 1회 표시됩니다.

### YouTube 연결 (최초 1회)
1. 최초 송출 시 브라우저가 열리면 **본인 Google 계정**으로 로그인.
2. 이후에는 자동 갱신되며 재로그인이 필요 없습니다.
3. 채널에 라이브 스트리밍이 활성화돼 있어야 합니다 (YouTube Studio → 라이브 스트리밍, 최초 활성화에 24시간 소요될 수 있음).

## 2. 일상 사용

- **PC를 켜면 끝** — 약 30초 후 자동으로 송출+녹화가 시작됩니다.
- 화면 좌측 상단의 **9px 점**이 상태를 보여줍니다:
  | 색 | 의미 |
  |---|---|
  | 🟢 | 정상 송출 중 |
  | 🟡 | 연결 중 |
  | 🔴 | 오류·재시도 중 (녹화는 계속됨) |
- 라이브는 항상 **일부 공개(Unlisted)** 로 생성됩니다. 링크를 아는 사람만 시청 가능.
- PC를 끄면 자동으로 방송이 종료되고 녹화 파일이 마감됩니다.

## 3. 제어판 (`Ctrl+Shift+F12`)

| 영역 | 기능 |
|---|---|
| 상태 배지 | LIVE / 연결 중 / 재시도 중 + 수동 시작·중지 |
| 성능 | 업로드 kbps, fps, CPU/GPU |
| 오디오 | 시스템·마이크 독립 볼륨, 마이크 장치 선택 |
| 자원 제한 | 25/50/75/제한 없음 (근사 상한, 다음 세션부터) |
| 녹화 | 현재 파일명, 누적 용량, 남은 디스크 |
| 로그 | 최근 1000줄 실시간 |
| 설정 | 단축키 변경, 자동 시작 방식, 녹화 폴더/한도/보존일 |

## 4. 녹화 파일

- 위치: 설치 시 지정한 폴더. 파일명 `SilentStream_REC_2026-06-12_0930.mp4`.
- 부팅~종료가 1개 파일입니다 (인코더 재시작 시 `_part2` 파일 추가).
- **7일**(설정 가능) 지난 파일과 **용량 한도**(기본 100GB) 초과분은 오래된 것부터 자동 삭제됩니다.
- 송출이 끊겨도 녹화는 계속됩니다. 갑자기 전원이 꺼져도 파일은 재생 가능합니다.

## 5. 문제 해결

| 증상 | 확인 사항 |
|---|---|
| 박스가 계속 🟡 | 인터넷 연결, YouTube 라이브 스트리밍 활성화 여부 |
| 박스가 계속 🔴 | `%AppData%\SilentStream\logs\` 최신 로그 확인. 자동으로 무한 재시도합니다 |
| 브라우저 로그인 안 뜸 | `%AppData%\SilentStream\client_secret.json` 존재 확인 |
| 마이크 소리 없음 | 제어판 → 오디오 → 장치 선택, 볼륨 슬라이더 확인. 마이크가 없어도 시스템 소리는 송출됩니다 |
| 녹화만 되고 송출 안 됨 | 정상 동작입니다(네트워크 문제 시). 복구되면 자동 재연결 |
| 게임 전체화면에서 검은 화면 | 자동 재초기화를 기다리거나 창 모드(테두리 없는 전체화면) 사용 |
| 단축키가 안 먹힘 | 다른 프로그램과 충돌 — 설정에서 다른 조합으로 변경 |

- 로그: `%AppData%\SilentStream\logs\SilentStream_날짜.log` (180일 보관)
- 설정: `%AppData%\SilentStream\config.json`

## 6. 제거

제어판 → 앱 → SilentStream 제거. 자동 시작 등록은 함께 해제되며,
**녹화 파일과 설정은 삭제되지 않습니다** (직접 삭제 가능).
